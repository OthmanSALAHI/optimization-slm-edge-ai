\chapter{Data Preparation}

\vspace*{0.3cm}

Data preparation converts the original instruction-following dataset into a
format suitable for fine-tuning \texttt{google/flan-t5-small}. The model expects
text inputs and text outputs, so each example must be transformed into a clear
prompt-answer structure.

\section{Dataset and Configuration}

The dataset used is \texttt{yahma/alpaca-cleaned}, downloaded with the Hugging
Face \texttt{datasets} library. Each example contains an instruction, an optional
input, and an output.

\section{Processing Steps}

The script \texttt{src/prepare\_data.py} loads the configuration, downloads the
dataset, shuffles it, creates the train/validation/test subsets, cleans empty
fields, converts examples to instruction-tuning format, and saves the processed
files.

\section{Instruction-Tuning Format}

The preprocessing converts the original fields into:

\begin{itemize}
    \item \texttt{source\_text}: prompt given to the model.
    \item \texttt{target\_text}: expected answer.
\end{itemize}

With input, the prompt format is:

\begin{lstlisting}[style=jsonStyle]
Instruction: <instruction>
Input: <input>
Answer:
\end{lstlisting}

Without input, the format is:

\begin{lstlisting}[style=jsonStyle]
Instruction: <instruction>
Answer:
\end{lstlisting}
